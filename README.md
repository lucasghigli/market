🛒 FreshMart Supermarket — React.js E-Commerce Website

FreshMart Supermarket is a modern, responsive supermarket e-commerce web app built with React.js, Vite, and React Router. It simulates a complete online grocery shopping experience, including product browsing, category filtering, shopping cart management, checkout flow, user authentication, order history, and an admin dashboard for managing products, users, orders, and categories.

The project focuses on creating a realistic full-featured frontend application using localStorage for data persistence, protected routes for user/admin areas, and a clean responsive UI that works across desktop and mobile devices.

✨ Features:
- 🥦 Product catalog with categories, search, sorting, and filtering
- 🛒 Persistent shopping cart using localStorage
- 💳 Simulated checkout and order creation
- 🔐 Login/register system with protected routes
- 👤 User dashboard with order history
- 🧑‍💼 Admin panel for products, orders, users, and categories
- 📱 Fully responsive design for desktop and mobile
- ⚡ Built with Vite for fast development and optimized builds

🛠️ Tech Stack:
React.js, React Router, Vite, JavaScript, CSS, localStorage

🚀 A great portfolio project showing frontend architecture, routing, state management, e-commerce logic, responsive design, and admin/user workflows.
